# react-native-rtl-starter

React Native starter template pre-configured for RTL languages and custom fonts.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/react-native-rtl-starter.git
cd react-native-rtl-starter
```
