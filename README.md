## 🌟 Overview
Save days of configuration headaches. This React Native template is strictly configured for Right-To-Left (RTL) languages like Persian and Arabic. It includes custom font loading and forced RTL layout management via I18nManager.

## ✨ Key Features
- 🇮🇷 **Forced RTL**: I18nManager pre-configured to ensure RTL layout.
- 🔤 **Custom Fonts**: Persian fonts linked and ready to use in styles.
- ⚛️ **Modern React**: Hooks and functional components standard.

## 🚀 Getting Started

### Prerequisites
Make sure you have `React_Native` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/react-native-rtl-starter.git
cd react-native-rtl-starter
npm install && cd ios && pod install
```

### Usage
```bash
npx react-native run-android
```
